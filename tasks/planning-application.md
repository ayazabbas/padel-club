# Planning Application

**Status:** ⏳ Not Started  
**Target:** Q2-Q3 2026  
**Dependencies:**
- ⏳ Lease signed (need legal interest in property)
- ⏳ Pre-app feedback received (informs application)
- ⏳ Fit-out design complete (plans needed for application)

---

## Objective

Obtain planning permission for change of use to Class E(d) (indoor sport) and any associated works (lighting, signage, alterations).

---

## What Requires Planning Permission?

| Item | Likely Needed? | Notes |
|------|----------------|-------|
| Change of use to sport | Maybe | If already Class E, may not need |
| External alterations | Yes | New doors, loading bay modifications |
| Floodlighting | Yes | Most contentious issue |
| Signage | Yes | Separate advertisement consent |
| Parking changes | Maybe | If layout changing |
| Operating hours | Condition | Likely restricted (e.g., close by 22:00) |

See [Luton Planning Guide](../research/luton-planning-guide) for full details.

## Pre-Application (Do Before Lease if Possible)

### 1. Prepare Pre-App Submission
- [ ] Site location plan (1:1250 or 1:2500)
- [ ] Block plan (1:500 or 1:200)
- [ ] Description of proposed use
- [ ] Proposed operating hours
- [ ] Parking/access arrangements
- [ ] Any external changes

### 2. Submit Pre-Application
- [ ] Use [Luton online form](https://myforms.luton.gov.uk/service/Apply_for_pre_application_development)
- [ ] Pay fee (check current rates)
- [ ] Allow 6-8 weeks for response

### 3. Review Pre-App Response
- [ ] Note any concerns raised
- [ ] Understand conditions likely to be imposed
- [ ] Identify if any issues are deal-breakers
- [ ] Adjust plans if needed

---

## Full Planning Application (After Lease Signed)

### 4. Appoint Planning Consultant (Optional but Recommended)
- [ ] Get quotes from local planning consultants
- [ ] They can prepare and manage application
- [ ] May improve chances of approval
- [ ] **Budget:** £1-3k

### 5. Prepare Application Documents
- [ ] **Required drawings:**
  - Site location plan (red line boundary)
  - Existing floor plans
  - Proposed floor plans
  - Elevations (if external changes)
- [ ] **Supporting documents:**
  - Design & Access Statement
  - Noise Impact Assessment (likely required)
  - Transport Statement (if >10 staff or significant traffic)
  - Lighting Assessment (if floodlighting)
- [ ] **Application form** via Planning Portal

### 6. Submit Application
- [ ] Submit via [Planning Portal](https://www.planningportal.co.uk/)
- [ ] Pay application fee (~£586 for change of use as of April 2025)
- [ ] Note validation requirements — 70% of Luton apps rejected at first submission
- [ ] Track application reference number

### 7. Application Period
- [ ] Neighbour consultation (21 days)
- [ ] Officer assessment (8 weeks minor / 13 weeks major)
- [ ] Respond to any queries from planning officer
- [ ] May need to provide additional information

### 8. Decision
- [ ] If **approved**: Review all conditions carefully
- [ ] If **refused**: Consider appeal or revised application
- [ ] If **conditions unacceptable**: Negotiate or appeal specific conditions
