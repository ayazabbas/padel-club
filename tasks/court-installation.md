# Court Installation

**Status:** ⏳ Not Started  
**Target:** Q3-Q4 2026  
**Dependencies:**
- ⏳ Lease signed (access to property)
- ⏳ Planning permission granted (or confirmation not needed)
- ⏳ Funding in place (courts need paying for)
- ⏳ Fit-out design finalised

---

## Objective

Install 4 indoor padel courts plus supporting infrastructure (lighting, flooring, reception, changing rooms).

---

---

## Once Lease is Signed

### 3. Site Survey
- [ ] Arrange for court supplier to survey property
- [ ] Confirm floor condition
- [ ] Finalize court layout (orientation, spacing)
- [ ] Identify any site-specific issues

### 4. Design Fit-Out
- [ ] **Court layout** — 4 courts, orientation, spectator areas
- [ ] **Reception area** — Check-in, seating, display
- [ ] **Changing facilities** — Toilets, showers, lockers
- [ ] **Storage** — Equipment, cleaning supplies
- [ ] **Lighting** — LED sports lighting (may be included with courts)
- [ ] **Heating/ventilation** — Consider player comfort
- [ ] **Flooring** — Outside court areas

### 5. Appoint Contractors
- [ ] **Court supplier** — Confirmed with deposit
- [ ] **General contractor** — For non-court fit-out work
- [ ] **Electrician** — Certified for commercial work
- [ ] **Plumber** — For changing facilities
- [ ] Get multiple quotes for each trade

---

## Installation Phase

### 6. Prepare Site
- [ ] Clear property of any debris/existing fixtures
- [ ] Ensure floor is clean and level
- [ ] Confirm electrical supply adequate (may need upgrade)
- [ ] Install any required fire safety measures

### 7. Court Installation
- [ ] Supplier delivers court components
- [ ] Installation team on-site (typically 1-2 weeks per court)
- [ ] **Check during installation:**
  - Courts level and square
  - Glass panels secure
  - Turf properly laid
  - Lighting adequate (min 300 lux recommended)
- [ ] Final inspection with supplier
- [ ] Sign off on installation
- [ ] Obtain warranty documentation

### 8. Supporting Fit-Out
- [ ] Reception desk and seating
- [ ] Changing room fit-out
- [ ] Signage (internal and external)
- [ ] Fire safety equipment
- [ ] First aid facilities
- [ ] Cleaning equipment storage

### 9. Systems & Equipment
- [ ] Install booking system (custom build or provider)
- [ ] Point of sale system (if selling equipment/refreshments)
- [ ] CCTV/security
- [ ] Access control (if 24/7 operation planned)
- [ ] Wi-Fi installation
- [ ] Purchase initial equipment stock:
  - Rental rackets (20-30)
  - Balls (bulk purchase)
  - Cleaning equipment
