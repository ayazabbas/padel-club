# Legal & Regulatory Requirements — Research

*Last Updated: 3 February 2026*

## Summary

- **Planning:** Indoor sports = Use Class E(d). No permission needed if property already Class E.
- **Key licenses:** Premises (if alcohol), PPL PRS (music), food hygiene (if café)
- **Insurance:** Public liability essential, employers liability if staff
- **No padel-specific regulations** — LTA is governing body

---

## Planning Permission

### Use Classes

| Class | Description | Notes |
|-------|-------------|-------|
| **E(d)** | Indoor sport, recreation, fitness | Target class for padel |
| B2 | Industrial | Would need change of use |
| B8 | Storage/Distribution | Would need change of use |

### Key Points

- Indoor sports facility = Use Class E(d)
- If property already Class E → no planning permission needed
- B2 or B8 → Class E requires planning application
- Outdoor courts may need separate consideration

### Luton Council

- Planning enquiries: 01582 546 000
- Website: luton.gov.uk
- Pre-application advice recommended

## Padel-Specific

- No specific UK regulations for padel
- LTA is the governing body
- Courts should meet FIP/ITF specifications
- Consider LTA registration for credibility and potential discounts
